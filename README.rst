Overview
========

A Blog implementation for Plone. 
Features:
- Tags
- Categories
- Archiv

No Workflow provided. 

Install
======

- Add ``ftw.blog`` to yout buidlout configuration

:: 
  [instance]
  eggs = 
    ftw.blog
    
- Run buildout

- Install ``ftw.blog`` in portal_setup
